# VoltSafe AI

## About
VoltSafe AI is a smart electrical hazard detection and prevention system designed to reduce accidents caused by water contact and unsafe electrical conditions.

## Problem
Electrical accidents caused by leakage current, exposed wires, and water contact can lead to severe injuries and deaths.

## Solution
VoltSafe AI detects dangerous conditions using sensors and automatically alerts users and cuts power using a relay module.

## Features
- Water detection
- Hazard alerts
- Automatic power cutoff
- Buzzer warning system
- AI-based monitoring concept

## Components Used
- ESP32 / ESP8266
- Water Sensor
- Relay Module
- Buzzer
- LEDs
- Jumper Wires
